# 高分辨率投屏

TeamhelperDeveloperTools支持超高分辨率投屏（建议使用USB接口），同时可以对投屏窗口一键置顶、一键全屏，解决了第三方投屏工具投屏质量低、速度慢的问题，同时支持屏幕触控、虚拟方向键控制安卓设备。同时投屏面板可联动应用面板，双击打开应用，更多功能请自行探索。
