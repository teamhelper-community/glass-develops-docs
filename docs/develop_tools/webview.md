# WebView调试

网页面板功能同 `chrome://inspect/#devices` 一致，可以检测到设备上开启了 debug 模式的 webview 组件，对其使用 chrome 调试器进行调试。该功能一般用于调试安卓手机端WebView内嵌页面，调试JSBridge。

:::warning

目前仅能检测到正在使用中应用内的 webview 页面。

:::
